## Install

```sh
$ npm install -D @markuplint/vue-spec

$ yarn add -D @markuplint/vue-spec
```

## Usage

Add `specs` option into your [configuration file](https://markuplint.dev/configuration#parser).

```json
{
  "specs": {
    ".vue$": "@markuplint/vue-spec"
  }
}
```
