## Install

```sh
$ npm install -D @markuplint/react-spec

$ yarn add -D @markuplint/react-spec
```

## Usage

Add `specs` option into your [configuration file](https://markuplint.dev/configuration#parser).

```json
{
  "specs": {
    ".[jt]sx?$": "@markuplint/react-spec"
  }
}
```
