## Install

```sh
$ npm install -D @markuplint/pug-parser

$ yarn add -D @markuplint/pug-parser
```

## Usage

Add `parser` option into your [configuration file](https://markuplint.dev/configuration#parser).

```json
{
  "parser": {
    ".(?:pug|jade)$": "@markuplint/pug-parser"
  }
}
```
